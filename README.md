# 🎮 JustPlayNoww

> **Play free browser games instantly — no downloads, no login, no hassle.**

A modern, open-source game portal built with pure HTML, CSS & JavaScript. Featuring a premium dark-themed UI with glassmorphism cards, animated particle backgrounds, and 12 curated games you can play right in your browser.

## 🎯 Features

- 🌙 **Premium Dark Theme** — Glassmorphism cards with neon glow effects
- ✨ **Animated Background** — Interactive particle canvas with floating connections
- 🔍 **Search & Filter** — Find games instantly by name or category
- 🏷️ **Category System** — Games organized as Arcade, Puzzle, Classic, Skill & Built-in
- 📱 **Fully Responsive** — Looks great on desktop, tablet, and mobile
- ⚡ **Instant Play** — Games load in an embedded iframe with loading indicator
- 🚫 **No Dependencies** — Pure HTML, CSS & JavaScript — zero build tools needed

## 🚀 Getting Started

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/krishna7805/JustPlayNoww.git
   cd JustPlayNoww
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   open index.html
   ```
   Or use a live server (e.g., VS Code Live Server extension) for the best experience.

### Deploy to GitHub Pages

1. Go to your repo's **Settings → Pages**
2. Set source to **Deploy from a branch**
3. Select **main** branch and **/ (root)** folder
4. Click **Save** — your site will be live at `https://<username>.github.io/JustPlayNoww/`

## 🤝 Contributing

Contributions are welcome! Here's how you can help:

1. **Fork** the repository
2. **Create** a new branch (`git checkout -b feature/new-game`)
3. **Add** your game in the `HTML/` folder (single self-contained HTML file)
4. **Add** a thumbnail in the `img/` folder
5. **Update** `index.html` to include your game card
6. **Commit** and **push** your changes
7. Open a **Pull Request**

### Game Guidelines

- Games should be **self-contained** in a single HTML file (inline CSS + JS)
- Use a **dark theme** to match the portal aesthetic (background: `#0a0a1a`)
- Make games **responsive** and **mobile-friendly**
- No external dependencies (CDNs are okay for fonts)
